# VS Code V#

[![Visual Studio Marketplace Version](https://img.shields.io/visual-studio-marketplace/v/)](https://marketplace.visualstudio.com/items?itemName=)

Provides syntax highlighting and use of the V# Language Server, which itself provides formatting, goto-definition, autocompletion, and more!

This extension requires the V# compiler to be installed. If you don't already have the V# compiler on your system, please follow the instructions at <> before install this extension.
